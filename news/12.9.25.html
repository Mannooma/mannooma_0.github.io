MINECRAFT 1.21.11 RELEASE CANDIDATE 3 DROPS!

Three critical fixes for Java Edition are here. Get ready to mine!

Written By: Mannooma  |   Published: 12/9/25

Hello, Minecrafters! The developers at Mojang just shipped **Release Candidate 3 for 1.21.11**! Yes, you heard that right—another candidate version is out, and it's addressing three crucial issues that cropped up right after the last update. This is a big deal for Java Edition players!


It looks like the team is pushing hard for stability before the full launch. This Release Candidate focuses purely on bug squashing, ensuring our experience is smooth and polished. Ready to see what made the cut? Let's dive into the patch notes!


This update is a Release Candidate for Minecraft: Java Edition. It's available now in your launcher, but remember to back up your worlds!

Three Key Bugs Smashed in RC 3!

The official notes confirm three critical fixes that have been addressed in this new Release Candidate.
It’s exciting to see these issues resolved so quickly!

Here are the issues we can finally say goodbye to in 1.21.11 Release Candidate 3:

Fixed Bugs List:

  • [MC-304790]: Transparent item models are rendered incorrectly. *Say goodbye to weird-looking blocks!*

  • [MC-304797]: Zombie nautiluses generated in ocean ruins are not persistent. *Now your spooky ocean friends will actually stick around.*

  • [MC-304810]: Zombies, skeletons and zombie horses can catch fire from light sources when exposed to the sky in an arid biome. *That was a strange one—mobs in arid biomes can chill out now.*

How to Get the Release Candidate NOW!

If you're eager to try out these fixes (and I know you are!),
the Release Candidate is available for **Minecraft: Java Edition**.

To Install: Head over to the Minecraft Launcher and make sure you enable "snapshots" in the **"Installations"** tab!

**IMPORTANT NOTE:** Testing versions like this can corrupt your world! Please, please, remember to back up your save files or run this in a separate folder from your main worlds. Safety first!
